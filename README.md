# FrameFit AR prototype

Рабочий PWA-прототип для примерки глубоких фоторамок через камеру телефона.

## Что умеет
- открывается по ссылке на телефоне;
- запускает камеру;
- показывает виртуальную глубокую рамку поверх объекта;
- дает выбрать размер рамки и глубину;
- позволяет двигать рамку пальцем;
- проверяет, подходит ли объект по глубине;
- работает как PWA, можно добавить на экран телефона.

## Как запустить
1. Загрузить папку на любой HTTPS-хостинг: Netlify, Vercel, GitHub Pages, Timeweb, свой сайт.
2. Открыть `index.html` с телефона.
3. Разрешить доступ к камере.

Важно: камера в браузере работает только по HTTPS или на localhost.

## Что доработать для продаж на Ozon
- добавить реальные размеры всех SKU;
- добавить кнопки `Купить на Ozon` для каждой рамки;
- добавить сохранение снимка с наложенной рамкой;
- добавить калибровку по листу А4, банковской карте или линейке;
- добавить WebXR-режим для Android;
- сделать QR-код в карточку товара: `Проверьте размер перед покупкой`.

## Важно при обновлении
После изменений увеличивайте версию `CACHE_NAME` в `sw.js`, например:
`framefit-ar-v1` → `framefit-ar-v2`.
Иначе телефон может показывать старую версию из кэша.
